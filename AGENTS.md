# AGENTS.md

## 项目任务

本仓库用于完成一篇中文文献综述，主题为：机制转移、多边主义、瓦森纳协定、小多边机制与中美关系。

## 工作原则

1. 不要编造论文内容。
2. 如果某篇 PDF 无法读取或内容不完整，必须明确说明。
3. 所有输出使用中文。
4. 不要直接写终稿，必须先完成文献笔记和文献矩阵。
5. 文献综述不能逐篇罗列，要按照主题与文献层次（包括体系层次、国家层次、双边层次、单元层次）组织。
6. 语言要学术化，但不要过于生涩。注意中文表达，逻辑清晰，适合本科课程论文使用。
同时注意：解释力对比不要太主观，文献所属层次分为单元国家双边体系层次，而且双边和单元层次不是递进关系，而是并列关系，制度复杂性与霸权认知衰退不是递进关系
7. 保留论文标题、作者或关键词提示，方便后续补充引用格式。

## 执行顺序

1. 检查 papers 文件夹中是否有八篇论文。
2. 为每篇论文生成一个 Markdown 文献笔记，放入 literature_notes 文件夹。
3. 每篇笔记包括：
   - 文章标题，期刊年份作者
   - 研究问题
   - 核心观点、核心变量
   - 关键词
   - 理论框架、知识脉络
   - 论证逻辑与结果
   - 可借鉴之处
4. 生成 literature_matrix.md，对八篇文献进行横向比较。
5. 生成 literature_review_outline.md，按主题组织综述框架。
6. 最后生成 literature_review_draft.md，形成 2000 字左右中文初稿。

## 禁止事项

- 不要虚构不存在的文献观点。
- 不要把文献综述写成八篇论文的读书笔记拼接。
- 不要忽略机制转移、多边主义、瓦森纳协定、小多边、中美关系之间的理论联系。
- 不要在没有充分阅读文献的情况下直接生成终稿。
